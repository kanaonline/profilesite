    • Skill

    • HTML

      構造的にマークアップし、SEO対策を考慮した設計を行います。

    • CSS

      アニメーション作成、レスポンシブ対応、またSCSSを用いて効率的にコーディングを行います。

    • JavaScript

      JavaScriptを用いて様々な機能の実装を行います。
      (ex)ハンバーガーメニュー、ドロワーメニュー、スライダー、アコーディオン etc..
      またjQueryを用いての実装も可能です。

    • WordPress

      インストールなどの環境構築、既存テーマのカスタマイズ制作や修正を行えます。
      また、カスタム投稿やカスタムフィールドなどを用いて、オリジナルテーマの制作が可能です。

    • PHP

      PHPを用いることで、WordPressの設計を自由に行えますので、 お客様のイメージ通りのWebサイトを構築できます。
      またPHPを用いることにより動的なWebサイト運営を行い、お客様自身で簡単に更新できます。

    Service

    • ホームページ制作

      お客様のご希望に沿ったホームページを制作致します。
      HTML/ CSS(SCSS) /JavaScriptを使用した静的なホームページやWordPressを使用したお客様自身が更新できる動的なホームページを制作致します。
      また様々なデバイス(パソコン・タブレット・スマートフォン)へのレスポンシブデザインにも対応致します。

    • ホームページリニューアル

      現状のサイトの改善点を改善することはもちろんのこと、よりお客様の魅力が伝わるホームページにリニューアル致します。
      またHTMLやCSSを使用した静的サイトからWordPressを使用した動的サイトへの移行も対応致します。
      昔に制作したタブレットやスマートフォンに対応していないホームページのレスポンシブデザインに対応致します。

    • その他

      品質確認

      デザインデータと相違がないか、ホームページの表示・挙動に問題はないか等、細部まで品質確認を行います。

      適宜、報告・連絡・相談を行い、共有情報に差異はないか、問題がないか等確認し、円滑な業務遂行を行います。

      迅速な対応

      確認事項や疑問点などに対し、即日迅速なレスポンスを行います。
